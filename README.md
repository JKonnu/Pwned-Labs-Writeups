## Overview of the Repository
This repository contains detailed write-ups, notes, and walkthroughs for labs completed on the Pwned Labs platform. The focus of these labs is cloud security, AWS exploitation techniques, IAM enumeration, privilege escalation, misconfiguration discovery, and defensive awareness within cloud environments.

### Each lab write-up contains:
- A high-level overview and key concepts
- Technical attack walkthroughs
- Enumeration methodology
- Exploitation techniques
- Detection opportunities
- Mitigation recommendations
- Lessons learned and key takeaways
<br>
The purpose of this repository is to document practical cloud security learning, reinforce methodology, and build a professional portfolio demonstrating offensive and defensive cloud security skills.

## Platform Info:
Pwned Labs is a hands-on cloud security learning platform focused on practical attack and defense scenarios across multiple cloud and container technologies. The platform provides real-world style labs designed to simulate common cloud misconfigurations, privilege escalation paths, identity abuse, insecure permissions, and security weaknesses found in modern environments.

The platform currently offers labs and learning content focused on:
- AWS Security
- Azure Security
- GCP Security
- Kubernetes Security
The labs are designed to provide practical experience with both offensive and defensive cloud security techniques while reinforcing real-world security assessment methodology.
